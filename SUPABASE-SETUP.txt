SUPABASE SETUP — THE 615 HIDEAWAY MEMBER JOIN
=============================================

This gets https://www.the615hideaway.com/join working so fans can create free accounts.

STEP 1 — Create Supabase project (5 min)
----------------------------------------
1. Go to https://supabase.com/dashboard
2. New project → name: the615hideaway
3. Choose a strong database password (save it)
4. Region: East US (closest to Nashville)

STEP 2 — Run the database schema
--------------------------------
1. Supabase Dashboard → SQL Editor → New query
2. Paste everything from: supabase/schema.sql
3. Click Run
4. Run these migrations in order:
   - supabase/migration-member-type.sql
   - supabase/migration-artists.sql
   - supabase/migration-dj-profiles.sql

STEP 3 — Auth settings
----------------------
1. Authentication → URL Configuration
2. Site URL: https://www.the615hideaway.com
3. Redirect URLs (add ALL of these — wildcards fix confirm-email links):
   - https://www.the615hideaway.com/**
   - https://the615hideaway.com/**
   - https://the615hideaway.vercel.app/**

   If wildcards don't work in your dashboard, add each line:
   - https://www.the615hideaway.com/account
   - https://www.the615hideaway.com/artist-portal
   - https://the615hideaway.com/account
   - https://the615hideaway.vercel.app/account

4. Authentication → Providers → Email
   - Keep Email enabled
   - For fastest launch: turn OFF "Confirm email" (optional — turn back on later)
   - Or leave confirm ON and users verify email before first sign-in

STEP 4 — Add keys to Vercel
---------------------------
1. Supabase → Project Settings → API
2. Copy:
   - Project URL  → SUPABASE_URL
   - anon public key → SUPABASE_ANON_KEY

3. Vercel → the615hideaway project → Settings → Environment Variables
4. Add these for Production (and Preview if you want):
   - SUPABASE_URL
   - SUPABASE_ANON_KEY
   - RADIO_APPS_SCRIPT_URL (same URL as radio-dj/js/config.js googleScriptUrl)
   - RADIO_SERVICE_KEY (generate a long random secret — must match Apps Script)

5. Apps Script → Project Settings → Script properties → add RADIO_SERVICE_KEY
6. Paste radio-dj/google-apps-script/supabase-bridge.gs into Code.gs and add the doPost handler
7. Redeploy Apps Script (new version) and Vercel site

STEP 5 — Test
-------------
1. Visit https://www.the615hideaway.com/join
2. Create a test account
3. Should land on /account with your name and email
4. Supabase → Authentication → Users should show the new member
5. Supabase → Table Editor → profiles should have a matching row
